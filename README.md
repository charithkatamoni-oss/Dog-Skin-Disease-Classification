# Dog Skin Disease Classification

## Project Description
This project aims to classify dog skin diseases using deep learning techniques. The model is trained on image datasets to identify different skin disease categories in dogs.

## Dataset
Dataset Source: Kaggle Dog Skin Disease Dataset

The dataset contains labeled images of dogs with different skin conditions and healthy skin samples.

## Models Used
- Custom CNN
- MobileNetV2

## Results
The model was trained and evaluated on the dataset and achieved satisfactory classification performance.

## Requirements
- Python 3.x
- TensorFlow
- Keras
- NumPy
- Matplotlib
- Scikit-learn

## How to Run
1. Download the dataset.
2. Open the notebook file.
3. Install the required libraries.
4. Update the dataset path if necessary.
5. Run all notebook cells sequentially.
